# Symfony - Vue SPA example

### Stack
#### - PHP 8.2
#### - Vue 3.3
#### - Vuetify 3.3.4
#### - Node 16.20.1
#### - Symfony 6.3

### Build and Run

```bash
make
```

### Login Form (admin0/12345 or user0/12345)
```bash
http://127.0.0.1:8051/
```

### Stop containers

```bash
docker-compose down
```
